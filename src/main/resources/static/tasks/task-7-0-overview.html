Create and Deploy Connectors


Most Composable Commerce solutions involve integrating to external services, for example OMS for order processing and PSP for secure payment processing. By employing APIs to connect these services, businesses can ensure real-time data flow and synchronization, enhancing the overall customer experience. Lifestyle and Home Corp would like to create a Connector to implement their customer signup requirements.

Connector Applications



Start with the template to complete the following applications and test them locally.

  1. Customer Number Generator - A customer number is generated for each customer.
  2. Email Verification Token Generator - An email verification token is generated when a new customer signs up.
  3. Verify Email App - An endpoint that can be called to verify customer emails with a valid token.

Perform the following programmatically, using the SDK:
  • Get the Postman collection.
  • Clone the connector template.
  • Follow the instructions from the README and develop and test applications locally.
  • Create a new Github repository and push your local code to it.
  • Create a new release in Github.


Perform the following steps in the Merchant Center or Postman:

  • Create a new Connector.
  • Publish the Connector
  • Deploy the Connector in your project.
