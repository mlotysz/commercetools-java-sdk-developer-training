Checkout Workflow

Carts in Composable Commerce serve a crucial role of binding everything together. The whole checkout process revolves around Carts. When implemented properly, carts can help with the conversion rate and thereby reduce the number of abandoned carts. Lifestyle and Home Corp understands the importance of Carts and would like to implement the following related to the Carts:

Customer should be able to browse through products from a store.

A new cart, if one doesn’t exist, is created when a customer clicks ‘Add to Cart’.

Steps


Perform the following programmatically, using the SDK:

  • Get all products available in the store through Product Selections.
  • Create an anonymous Cart for your Store and add products (LineItems) to the cart.
