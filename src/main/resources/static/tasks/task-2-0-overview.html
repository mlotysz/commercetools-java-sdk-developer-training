Product Search


You are implementing this solution for Lifestyle and Home Corp. The first set of requirements are:

As a user you should be able to search for products using a keyword, and filter them by attributes like color, finish as well as price range.

Steps


Perform the following programmatically, using the SDK:

  1. The user can search through products by keyword.
  2. The system retrieves the relevant products along with filter choices.
  3. The system displays the filtered search results based on user selections.
  4. The user can view the product details with an option to add the products to the cart.
